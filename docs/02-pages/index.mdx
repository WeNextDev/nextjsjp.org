Next.js 13以前では、Pages Routerは Next.js でルートを作成する主な方法でした。各ファイルをルートにマッピングする直感的なファイルシステムルーターを使用していました。Pages Routerは新しいバージョンの Next.js でもサポートされていますが、Reactの最新機能を活用するために、新しい [App Router](/docs/app) への移行をお勧めします。

このドキュメントのセクションは、Pages Routerを使用している既存のアプリケーション向けです。
