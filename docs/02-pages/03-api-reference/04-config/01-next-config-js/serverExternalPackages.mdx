[`bundlePagesRouterDependencies`](/docs/pages/api-reference/config/next-config-js/bundlePagesRouterDependencies) オプションによる自動バンドリングから特定の依存関係をオプトアウトします。

これらのページは、ネイティブの Node.js `require` を使用して依存関係を解決します。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```

Next.jsは、現在互換性に取り組んでいる[人気のあるパッケージの短いリスト](https://github.com/vercel/next.js/blob/canary/packages/next/src/lib/server-external-packages.json)を含んでおり、自動的にオプトアウトされています：
