---
title: Version 10
description: Next.jsアプリケーションをVersion 9からVersion 10にアップグレードする
slug: /docs/pages/building-your-application/upgrading/version-10
---

Version 9とVersion 10の間に破壊的な変更はありませんでした。

version 10にアップグレードするには、次のコマンドを実行します：

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **補足:** TypeScriptを使用している場合、`@types/react`と`@types/react-dom`も対応するバージョンにアップグレードしてください。
