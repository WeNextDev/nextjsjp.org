Node.js version 18より前のバージョンでは、Next.jsは自動的に[undici](/docs/architecture/supported-browsers#polyfills)で`fetch()`をポリフィルし、デフォルトで[HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)を有効にします。

サーバー側のすべての`fetch()`呼び出しでHTTP Keep-Aliveを無効にするには、`next.config.js`を開き、`httpAgentOptions`設定を追加します：

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
