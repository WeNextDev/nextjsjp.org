Next.jsはデフォルトで、すべてのページに対して [etag](https://en.wikipedia.org/wiki/HTTP_ETag) を生成します。キャッシュ戦略によっては、HTMLページのETag生成を無効にしたい場合があります。

`next.config.js` を開き、`generateEtags` オプションを無効にします：

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
