Next.js には **2つ**のコマンドラインインターフェース（CLI）ツールが付属しています：

- **`create-next-app`**: デフォルトテンプレートまたはパブリック GitHub リポジトリの[例](https://github.com/vercel/next.js/tree/canary/examples)を使用して、新しい Next.js アプリケーションを迅速に作成します。
- **`next`**: Next.js 開発サーバーを実行し、アプリケーションをビルドするなどの操作を行います。
