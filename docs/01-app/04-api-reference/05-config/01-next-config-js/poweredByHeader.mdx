デフォルトでNext.jsは`x-powered-by`ヘッダーを追加します。オプトアウトするには、`next.config.js`を開き、`poweredByHeader`の設定を無効にします：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
