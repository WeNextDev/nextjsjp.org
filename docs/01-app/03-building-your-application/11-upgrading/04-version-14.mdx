## 13から14へのアップグレード

Next.jsバージョン14に更新するには、お好みのパッケージマネージャーを使用して以下のコマンドを実行してください：

```bash filename="Terminal"
npm i next@next-14 react@18 react-dom@18 && npm i eslint-config-next@next-14 -D
```

```bash filename="Terminal"
yarn add next@next-14 react@18 react-dom@18 && yarn add eslint-config-next@next-14 -D
```

```bash filename="Terminal"
pnpm i next@next-14 react@18 react-dom@18 && pnpm i eslint-config-next@next-14 -D
```

```bash filename="Terminal"
bun add next@next-14 react@18 react-dom@18 && bun add eslint-config-next@next-14 -D
```

> **補足:** TypeScriptを使用している場合は、`@types/react`と`@types/react-dom`も最新バージョンにアップグレードしてください。

### v14の概要

- 最小Node.jsバージョンが16.14から18.17に引き上げられました。これは16.xがサポート終了（end-of-life）に達したためです。
- `next export`コマンドは`output: 'export'`設定に置き換えられ、削除されました。詳細については[ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports)をご覧ください。
- `ImageResponse`の`next/server`インポートが`next/og`に変更されました。インポートを安全かつ自動的に変更するための[コードモッド](/docs/app/building-your-application/upgrading/codemods#next-og-import)が利用可能です。
- `@next/font`パッケージは組み込みの`next/font`に置き換えられ、完全に削除されました。インポートを安全かつ自動的に変更するための[コードモッド](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)が利用可能です。
- `next-swc`のWASMターゲットが削除されました。
