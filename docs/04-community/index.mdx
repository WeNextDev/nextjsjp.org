週間500万ダウンロードを超える Next.js は、世界中の多くの開発者による活発なコミュニティを持っています。以下が、コミュニティに参加する方法です：

## コントリビューション

Next.js の開発に貢献する方法はいくつかあります：

- [ドキュメンテーション](/docs/community/contribution-guide)：改善の提案、または Next.js の使い方をユーザーに理解してもらうための新しいセクションの作成。
- [サンプル](https://github.com/vercel/next.js/tree/canary/contributing/examples)：新しいサンプルの作成や既存のサンプルの改善により、他のツールやサービスと Next.js の統合を支援。
- [コードベース](https://github.com/vercel/next.js/tree/canary/contributing/core)：基本的なアーキテクチャについて学び、バグ修正やエラーに貢献し、新機能を提案。

## ディスカッション

Next.js について質問したい場合や、他の人を助けたい場合は、会話に参加することを歓迎します：

- [GitHub ディスカッション](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## ソーシャルメディア

最新の情報は [Twitter](https://x.com/nextjs) でフォローし、Next.js の動画は [Vercel YouTube チャンネル](https://www.youtube.com/@VercelHQ) を購読してください。

## 行動規範

私たちは、包括的で歓迎的なコミュニティの創造を信じています。そのため、全メンバーに [行動規範](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md) の遵守をお願いしています。このドキュメントは、参加者の行動に関する私たちの期待を概説しています。これを読み、安全で敬意のある環境の維持にご協力ください。
